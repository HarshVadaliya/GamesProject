Games Project

Dice Game

This is a 2 player game. Click on refresh in the game and the player with bigger number in dice wins the game and if both the players have a same number then it is a draw. This game is powered by the JavaScript's Math.random() function.

Drum Kit

It's party time and you are the DJ. Enjoy the drums and start the party. DJ is on ready now you gather food,drinks and friends for the party. Use Keyboard or mouse and feel the sound by creating your own tune.

Simon Game

Simon game is all about testing your memory power. Press any key to start the game and remember the colour that is pressed by computer and then press the same colour. This game never ends but it ends when your memory power gives up. You can also try to remember the sound of a colour.
